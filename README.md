# EcoGuia - Motor Automático de Reputação Social

## Descrição do Projeto
O EcoGuia atua como um hub centralizador de dados verificados. O sistema foi projetado para processar registros de práticas sustentáveis individuais, aplicando regras de validação por meio de um Motor de Reputação. Ele converte ações ecológicas em indicativos numéricos auditáveis e progressão de marcos. O objetivo central é fornecer um mecanismo padronizado e confiável para quantificar, auditar e exibir o histórico de impacto ambiental de usuários comuns.

## Tecnologias Utilizadas
O Front-End foi desenvolvido com foco em modularidade e responsividade nativa, utilizando:
* **HTML5:** Semântica estrutural e organização em múltiplas páginas.
* **CSS3:** Variáveis globais (`:root`), Grid Layout, Flexbox, Media Queries (Mobile, Tablet e Desktop) e animações (`keyframes`).
* **JavaScript (Vanilla):** Manipulação do DOM (Document Object Model), lógica de componentes interativos (Accordion, Voltar ao Topo, Menu Hambúrguer) e de formulários.
